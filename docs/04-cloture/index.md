# Clôture

Le semestre touche à sa fin : dernière ligne droite avant le dépôt de votre portfolio.

## Checklist qualité finale

Avant de considérer votre site terminé, vérifiez point par point :

- [ ] Toutes les pages prévues dans l'arborescence sont en ligne et accessibles depuis le menu
- [ ] Aucun texte de remplissage ("Lorem ipsum") ne subsiste
- [ ] Tous les liens internes et externes fonctionnent
- [ ] La page "Mentions légales" est présente et accessible
- [ ] La mention RGPD apparaît bien à côté du formulaire de contact
- [ ] Le formulaire de contact a été testé avec un envoi réel
- [ ] Le CV est téléchargeable
- [ ] L'affichage est vérifié sur mobile, tablette et bureau
- [ ] La charte graphique (couleurs, typographies) est respectée sur toutes les pages
- [ ] Les corrections issues de la revue par les pairs ont été appliquées

!!! warning "Ne sous-estimez pas cette étape"
    Un détail qui traîne (lien mort, faute, formulaire qui ne part pas) donne une impression de travail bâclé, même si le reste du site est réussi.

## Dépôt du site

Déposez l'URL finale de votre portfolio selon les modalités communiquées en séance. Votre site sera évalué sur pièces, à partir d'une grille de critères (voir [Ressources](../ressources/index.md)).

## Bilan

Vous terminez ce semestre avec un site professionnel réellement en ligne, sous votre propre nom de domaine — une compétence directement valorisable pour vos recherches de stage/alternance et votre entrée sur le marché du travail.
