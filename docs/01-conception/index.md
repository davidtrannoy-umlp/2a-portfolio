# Phase 1 — Conception

Cette première phase pose les fondations de votre portfolio, avant toute mise en ligne. À l'issue de ces séances, vous devez avoir :

- un **brief de projet** clair (à qui s'adresse votre portfolio, quels objectifs professionnels)
- la liste des **réalisations à présenter**, avec une arborescence de site
- une **direction artistique** (moodboard, palette, typographie)

!!! tip "Pourquoi cette étape compte"
    Un portfolio conçu dans la précipitation se voit tout de suite. Le temps passé ici vous fera gagner beaucoup de temps en phase de réalisation.

**Pages de cette phase :**

- [Cadrage & contenus](cadrage-contenus.md)
- [Identité visuelle](identite-visuelle.md)

    - Moodboard + wireframe (11/10/26)
